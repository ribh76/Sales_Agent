# V1 Scope

SalesCompass V1 helps a founder or sales leader decide where to focus their go-to-market motion.

## Included

- Authentication for a small team demo.
- Company profile intake with two paths: known customer history and no customer history.
- ICP analysis pipeline that combines deterministic heuristics with an optional Anthropic call.
- Segment scoring across fit, urgency, reachability, deal quality, and confidence.
- Recommended ICP, disqualifiers, messaging angles, outreach variations, and a 30-day action plan.
- Human feedback capture on usefulness, confidence, and execution fit.
- Baseline versus agent evaluation for demo profiles.

## Success Criteria

- A user can create an analysis in under five minutes.
- The results page clearly explains why an ICP was recommended.
- Demo output is deterministic when no LLM key is configured.
- Evaluation makes the improvement over a generic baseline visible.
