# Demo Script

1. Open the dashboard and show the existing demo profiles.
2. Start an analysis for "Northstar Enablement".
3. Show the history/no-history switch and explain why the intake changes.
4. Submit the profile.
5. Walk through the diagnosis, ranked segments, recommended ICP, disqualifiers, and outreach.
6. Submit a human checkpoint rating.
7. Open Evaluation and compare the agent against the generic baseline.

## Talk Track

SalesCompass is not trying to replace sales judgment. It compresses the first messy strategy pass, makes the assumptions visible, and gives the team a sharper starting point for customer discovery.
